                                     ABOUT                                     
--------------------------------------------------------------------------------
- This program is designed to take audio input rom the user (either by recording it 
yourself or submitting .wav files) from the HTML interface and predict one of three 
possible categories of cat meows:

        - "annoyed (brushing)"
        - "feed me"
        - "attention/stress"
- This program can be accessed from the GitHub pages link:
        https://autoupdatingbsod.github.io/Kittranslator
       
- This application relies heavily on the ONNX Machine Learning API, which currently 
has no support for iOS at the moment.

- This project is a prototype. It is not designed to be fully accurate at the moment.
